Thingder

Item matching webapp

Web2py, Python, javascript, jQuery, HTML, CSS

Thingder was a group project that I worked on for UC Santa Cruz' web applications class. We used the web2py web application framework by recommendation of the professor and used jQuery along with AJAX alongside it to facilitate asynchronous calls to our backend

We wanted there to be a place like craigslist where college students who don't have much money can trade items with other students, rather than only relying on cash to get things we want or need. We also wanted to facilitate communication between potential traders so we build in a messaging system so students could work out details like where and when to meetup.
