Similar Music

Music Searching Webapp

HTML5, CSS3, VueJS, Webpack, Masonry

Similar Music is a webapp that lets users search for music artists they enjoy in order to discover other similar artists. The app utilizes the spotify web API as well as the wikipedia web API for artist descriptions.

I built this webapp independently in order to learn VueJS, a javascript framework that has similarities to AngularJS and ReactJS. I abided by the component MVVM architecture which is recommended when using best practices in order to build the app. This project also gave me the opportunity to work with ES6, the upcoming Javascript specification.

Thingder

Item matching webapp

Web2py, Python, Javascript, jQuery, HTML/CSS, Bootstrap

Thingder was a group project that I worked on for UC Santa Cruz' web applications class. We used the web2py web application framework by recommendation of the professor and used jQuery along with AJAX alongside it to facilitate asynchronous calls to our backend

We wanted there to be a place like craigslist where college students who don't have much money can trade items with other students, rather than only relying on cash to get things we want or need. We also wanted to facilitate communication between potential traders so we build in a messaging system so students could work out details like where and when to meetup.

Studie

Group study app

AngularJS, Javascript, Firebase, HTML/CSS, Materialize

Studie was a group project that I worked on for UC Santa Cruz Software Design class. We decided to utilize AngularJS on the web side of things as well as using firebase for a backend as a service, and since it made it easy to synchronize across platforms.

We created Studie as an app for students to quickly and easily find study partners in classes that they may not know too many people in. It allowed people to set the location on campus that a study session or meeting might occur, and the time when everyone would meet. We also built a realtime chat into the app using firebase so that groups could quickly coordinate with each other
